# GL & CO. Construcciones

Sitio corporativo en React + Vite + Tailwind CSS para GL & CO. Construcciones.

## Caracteristicas
- Landing de una sola pagina con scroll suave.
- Secciones: Inicio, Nosotros y Contacto.
- Carrusel animado.
- Estilo corporativo tipo Apple con tarjetas tipo azulejo.
- Botones con animacion personalizada.
- Secciones oscuras con lineas abstractas.
- Integracion con Osisg y enlaces a App Store / Google Play.

## Scripts

```bash
npm install
npm run dev
```

## Estructura
- `src/pages/Home.tsx`: hero, carrusel y anuncio Osisg.
- `src/pages/About.tsx`: equipo, galeria y cobertura.
- `src/pages/Contact.tsx`: contacto y redes.

## Notas
- Las imagenes son de ejemplo y se pueden reemplazar.
- Actualiza email, WhatsApp y redes sociales en `src/pages/Contact.tsx`.
